Sample project to create AAR file with Android Studio to wrap your resources such as images...
 + Step 1: Open sample project MakeCustomNotificationIcon
 + Step 2: Go to liveopscustomnotificationicon module, add your images at res/drawable-xxx
 + Step 3: Build sample project. 
When you build your project the AAR file will automatically be assembled and placed in folder MakeCustomNotificationIcon\liveopscustomnotificationicon\build\outputs\aar
 
Reference: https://androidbycode.wordpress.com/2015/02/23/building-an-aar-library-in-android-studio/
